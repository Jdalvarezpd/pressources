## 📖 Sobre el Proyecto

**Pressources** es una plataforma tipo red social diseñada para resolver un problema de comunicación específico: conectar a **periodistas** con **fuentes de información**. 

La aplicación funciona como un puente digital donde los profesionales de la comunicación que buscan expertos, testimonios o información de primera mano, pueden encontrar y contactar fácilmente a personas dispuestas a hablar sobre temas o nichos específicos.

## 🚀 Características Principales

* **Perfiles Especializados:** Creación de cuentas diferenciadas para Periodistas (quienes buscan información) y Fuentes (quienes proveen información).
* **Directorio de Temas:** Clasificación de perfiles basada en áreas de especialidad, conocimientos o experiencias.
* **Interfaz Responsiva:** Diseño adaptable a diferentes dispositivos para facilitar la navegación.

## 🛠️ Tecnologías Utilizadas

Este proyecto fue construido desde cero sin el uso de frameworks pesados de backend, priorizando un desarrollo a medida:

* **Backend:** PHP nativo (Vanilla PHP).
* **Frontend:** HTML5, CSS3 y JavaScript puro (Vanilla JS) completamente a medida.
* **UI/Diseño:** Bootstrap (para el sistema de grillas y componentes base).
